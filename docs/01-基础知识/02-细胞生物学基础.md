# 02-细胞生物学基础

## 1. 定义与核心概念

### 1.1 细胞 (Cell)

**定义**：细胞是生物体结构和功能的基本单位，是生命活动的基本场所。

**核心概念**：
- **原核细胞**：无核膜包围的细胞核，如细菌
- **真核细胞**：有核膜包围的细胞核，如动植物细胞
- **细胞器**：细胞内具有特定功能的结构
- **细胞周期**：细胞从一次分裂结束到下一次分裂结束的过程

:::tip
细胞是生命的基本单位，了解细胞结构和功能是学习分子生物学的基础。
:::

## 2. 细胞结构

### 2.1 原核细胞结构

| 结构 | 功能 |
|------|------|
| 细胞壁 | 提供细胞形态和保护 |
| 细胞膜 | 控制物质进出 |
| 细胞质 | 细胞内代谢场所 |
| 拟核 | 含有遗传物质 DNA (Deoxyribonucleic Acid) |
| 核糖体 | 蛋白质合成场所 |

### 2.2 真核细胞结构

#### 2.2.1 细胞膜
- **结构**：磷脂双分子层，镶嵌蛋白质
- **功能**：物质运输、信号转导、细胞识别

#### 2.2.2 细胞核
- **结构**：核膜、染色质、核仁
- **功能**：储存遗传信息、控制基因表达

#### 2.2.3 细胞质细胞器

| 细胞器 | 结构 | 功能 |
|--------|------|------|
| 内质网 | 膜性管道系统 | 蛋白质合成与加工、脂质合成 |
| 高尔基体 | 扁平囊泡堆叠 | 蛋白质修饰、分类和运输 |
| 线粒体 | 双层膜结构 | 有氧呼吸、能量产生 |
| 叶绿体 | 双层膜结构（植物） | 光合作用 |
| 溶酶体 | 膜性囊泡 | 细胞内消化 |
| 核糖体 | 无膜结构 | 蛋白质合成 |
| 细胞骨架 | 蛋白质纤维网络 | 细胞形态维持、物质运输 |

## 3. 细胞周期

### 3.1 细胞周期阶段

1. **G1 期**（DNA 合成前期）
   - 细胞生长
   - 合成蛋白质和 RNA (Ribonucleic Acid)

2. **S 期**（DNA 合成期）
   - DNA 复制
   - 染色体复制

3. **G2 期**（DNA 合成后期）
   - 合成微管蛋白
   - 准备细胞分裂

4. **M 期**（分裂期）
   - 有丝分裂：核分裂
   - 胞质分裂：细胞质分裂

### 3.2 细胞周期调控
- **细胞周期检查点**：G1/S 检查点、G2/M 检查点、纺锤体检查点
- **细胞周期蛋白**：Cyclin
- **细胞周期蛋白依赖性激酶**：CDK (Cyclin-Dependent Kinase)

## 4. 细胞代谢

### 4.1 能量代谢
- **糖酵解**：细胞质中进行，产生 ATP (Adenosine Triphosphate)
- **柠檬酸循环**：线粒体中进行，产生 NADH (Nicotinamide Adenine Dinucleotide) 和 FADH₂ (Flavin Adenine Dinucleotide)
- **氧化磷酸化**：线粒体中进行，产生大量 ATP

### 4.2 物质代谢
- **蛋白质代谢**：合成、降解、氨基酸代谢
- **脂质代谢**：合成、降解、脂肪酸氧化
- **核酸代谢**：DNA 复制、RNA 合成

## 5. 细胞信号传导

### 5.1 信号传导途径
1. **受体激活**：信号分子与受体结合
2. **信号转导**：通过第二信使传递信号
3. **细胞响应**：基因表达变化、酶活性变化

### 5.2 主要信号通路
- **cAMP (Cyclic Adenosine Monophosphate) 通路**
- **MAPK (Mitogen-Activated Protein Kinase) 通路**
- **PI3K/Akt (Phosphoinositide 3-Kinase/Akt) 通路**
- **Notch 通路**
- **Wnt 通路**

## 6. 细胞分化与凋亡

### 6.1 细胞分化
- **定义**：细胞在发育过程中逐渐特化的过程
- **机制**：基因选择性表达
- **结果**：形成不同类型的细胞

### 6.2 细胞凋亡
- **定义**：程序性细胞死亡
- **特征**：细胞膜皱缩、染色质凝聚、DNA 片段化
- **意义**：维持组织稳态、清除受损细胞

:::warning
细胞凋亡是一种正常的生理过程，与细胞坏死不同，它是由基因调控的程序性死亡。
:::
