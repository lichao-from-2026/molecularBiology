# 01-PCR检测技术

## 1. 定义与核心概念

**PCR检测技术**（Polymerase Chain Reaction）是一种用于放大扩增特定的DNA片段的分子生物学技术。它能以极少量的DNA为模板，在体外快速扩增出大量的目标DNA片段，是分子生物学研究和临床诊断的重要工具。

## 2. 分子机制/原理

PCR技术的基本原理是：
1. **变性**（Denaturation）：在高温（94-98°C）下，双链DNA模板解旋为单链
2. **退火**（Annealing）：在低温（50-65°C）下，引物与单链DNA模板互补结合
3. **延伸**（Extension）：在中温（72°C）下，DNA聚合酶从引物3'端开始合成新的DNA链
4. **循环**：重复变性-退火-延伸过程25-40次，使目标DNA片段呈指数级扩增

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| DNA模板 | 待检测的DNA样品 |
| 引物（正向和反向） | 与目标DNA序列互补的寡核苷酸 |
| Taq DNA聚合酶 | 催化DNA合成的酶 |
| dNTP混合物 | 合成DNA的原料 |
| 10× PCR缓冲液 | 提供适宜的反应环境 |
| MgCl2溶液 | 影响酶活性和引物退火 |
| 无菌水 | 配制反应体系 |
| PCR仪 | 控制温度循环 |
| 琼脂糖凝胶 | 分离扩增产物 |
| 核酸染料 | 检测DNA |
| DNA分子量标准 | 确定扩增产物大小 |

## 4. 试剂配方

### 4.1 常用溶液配方

#### 4.1.1 10× PCR缓冲液
- **配方**：
  - 500 mM KCl
  - 100 mM Tris-HCl (pH 8.3 at 25°C)
  - 15 mM MgCl₂
- **配制方法**：
  1. 称取 3.73 g KCl，溶解于 80 mL 去离子水中
  2. 加入 1.21 g Tris base，调节 pH 至 8.3
  3. 加入 0.415 g MgCl₂·6H₂O
  4. 定容至 100 mL
  5. 过滤灭菌
  6. -20°C保存

#### 4.1.2 dNTP混合物（10 mM each）
- **配方**：
  - 10 mM dATP
  - 10 mM dTTP
  - 10 mM dCTP
  - 10 mM dGTP
- **配制方法**：
  1. 分别取等体积的100 mM dATP、dTTP、dCTP、dGTP
  2. 混合均匀
  3. 分装后-20°C保存

#### 4.1.3 1× TAE缓冲液
- **配方**：
  - 40 mM Tris-acetate
  - 1 mM EDTA (pH 8.0)
- **配制方法**：
  1. 称取 4.84 g Tris base，溶解于 800 mL 去离子水中
  2. 加入 1.14 mL 冰醋酸
  3. 加入 0.37 g EDTA·2H₂O
  4. 定容至 1 L
  5. 高压灭菌

#### 4.1.4 6× DNA上样缓冲液
- **配方**：
  - 0.25% bromophenol blue
  - 0.25% xylene cyanol FF
  - 30% glycerol
  - 10 mM EDTA
- **配制方法**：
  1. 称取 25 mg bromophenol blue，25 mg xylene cyanol FF
  2. 加入 30 mL 甘油
  3. 加入 0.37 g EDTA·2H₂O
  4. 定容至 100 mL
  5. 4°C保存

#### 4.1.5 琼脂糖凝胶
- **配方**：
  - 1-2% 琼脂糖
  - 1× TAE缓冲液
  - 0.5 μg/mL 核酸染料（如EB或SYBR Safe）
- **配制方法**：
  1. 称取 1-2 g 琼脂糖，加入 100 mL 1× TAE缓冲液
  2. 加热至琼脂糖完全溶解
  3. 冷却至约50°C，加入核酸染料
  4. 倒入凝胶模具中，插入梳子
  5. 室温凝固30分钟

## 5. 实验步骤

### 步骤1：PCR反应体系的配制

- **目的**：准备含有所有必要成分的PCR反应混合液
- **操作**：
  1. 在冰上配制反应体系（总体积25μL）：
     - 10× PCR缓冲液：2.5μL
     - MgCl2（25mM）：1.5-2.5μL（根据引物优化）
     - dNTP混合物（10mM each）：0.5μL
     - 正向引物（10μM）：1μL
     - 反向引物（10μM）：1μL
     - Taq DNA聚合酶（5U/μL）：0.2μL
     - DNA模板：1μL（约10-100ng）
     - 无菌水：补足至25μL
  2. 轻轻混匀，短暂离心
- **注意事项**：
  - 所有操作应在冰上进行，避免酶失活
  - 引物浓度和MgCl2浓度需要优化
  - 模板DNA应避免含有PCR抑制剂

### 步骤2：PCR循环参数设置

- **目的**：设置适合目标片段扩增的温度循环参数
- **操作**：
  1. 预变性：94-98°C，3-5分钟
  2. 循环（25-40次）：
     - 变性：94-98°C，30秒
     - 退火：50-65°C，30秒（根据引物Tm值调整）
     - 延伸：72°C，1分钟/1kb（根据目标片段长度调整）
  3. 最终延伸：72°C，5-10分钟
  4. 保存：4°C，短期保存
- **注意事项**：
  - 退火温度应根据引物的Tm值设置，通常比Tm值低5°C
  - 延伸时间应根据目标片段长度调整
  - 循环次数应根据模板浓度调整，避免过度扩增

### 步骤3：PCR产物的检测

- **目的**：验证PCR扩增是否成功，确定产物大小
- **操作**：
  1. 配制1-2%琼脂糖凝胶（含核酸染料）
  2. 取5-10μL PCR产物与上样缓冲液混合
  3. 上样到凝胶孔中，同时加入DNA分子量标准
  4. 120V电泳20-30分钟
  5. 在凝胶成像系统中观察结果
- **注意事项**：
  - 凝胶浓度应根据目标片段大小选择
  - 上样量应适当，避免条带过浓或过淡
  - 电泳时间应根据片段大小调整

### 步骤4：结果分析

- **目的**：判断PCR扩增是否成功，分析产物的特异性
- **分析要点**：
  1. **条带大小**：与预期大小是否一致
  2. **条带亮度**：是否有足够的扩增产物
  3. **非特异性条带**：是否存在杂带
  4. **阴性对照**：是否有污染
  5. **阳性对照**：是否正常扩增
- **注意事项**：
  - 应设置阳性对照和阴性对照
  - 如有非特异性条带，应优化PCR条件
  - 如无条带，应检查模板、引物或反应条件

## 5. 应用场景

- **基因克隆**：扩增目标基因片段用于克隆
- **基因检测**：检测特定基因的存在或缺失
- **基因突变分析**：检测基因的突变位点
- **基因表达分析**：半定量分析基因表达水平
- **病原体检测**：检测病毒、细菌等病原体
- **亲子鉴定**：分析DNA多态性
- **法医学鉴定**：分析DNA指纹

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **常规PCR** | 基本PCR技术 | 一般DNA扩增 |
| **实时定量PCR（qPCR）** | 实时监测扩增过程 | 基因表达定量 |
| **反转录PCR（RT-PCR）** | 以RNA为模板 | 分析基因转录水平 |
| **巢式PCR** | 两次扩增，提高特异性 | 低丰度模板检测 |
| **多重PCR** | 同时扩增多个目标 | 多基因检测 |
| **降落PCR** | 退火温度逐渐降低 | 提高扩增特异性 |
| **热启动PCR** | 热激活酶，减少非特异性扩增 | 复杂模板扩增 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无扩增产物 | 模板浓度低或质量差 | 增加模板量，提高模板质量 |
| 非特异性条带 | 退火温度低或引物特异性差 | 提高退火温度，重新设计引物 |
| 引物二聚体 | 引物浓度过高或引物互补 | 降低引物浓度，重新设计引物 |
| 产物量少 | 循环次数少或酶活性低 | 增加循环次数，使用新鲜酶 |
| 污染 | 环境或试剂污染 | 严格无菌操作，使用新试剂 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 灵敏度高，可检测极少量DNA | 容易受到污染，导致假阳性 |
| 特异性强，可扩增特定目标 | 扩增片段长度有限（一般<5kb） |
| 操作简单，自动化程度高 | Taq酶无3'→5'外切酶活性， fidelity较低 |
| 快速高效，2-3小时完成 | 需要已知目标序列设计引物 |
| 可应用于多种样品类型 | 对模板质量有一定要求 |
