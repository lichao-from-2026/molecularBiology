# 01-细胞培养技术

## 1. 定义与核心概念

**细胞培养技术**是指在体外模拟体内环境，培养和繁殖细胞的技术。哺乳动物细胞培养是分子生物学、细胞生物学和生物医学研究的基础技术，广泛应用于基因功能研究、药物筛选、疫苗生产等领域。

## 2. 分子机制/原理

细胞培养技术的基本原理是：
1. **无菌操作**：在无菌环境中操作，防止微生物污染
2. **培养基**：提供细胞生长所需的营养物质和生长因子
3. **培养条件**：控制温度、湿度、CO2浓度等环境因素
4. **细胞传代**：当细胞生长到一定密度时，进行传代培养
5. **细胞冻存与复苏**：保存细胞株，延长细胞寿命

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 细胞系 | 待培养的细胞 |
| 培养基（如DMEM、RPMI 1640） | 提供细胞生长所需的营养 |
| 血清（如胎牛血清） | 提供生长因子和营养 |
| 抗生素（青霉素/链霉素） | 防止细菌污染 |

| CO2培养箱 | 提供适宜的培养环境 |
| 倒置显微镜 | 观察细胞生长状态 |

| 移液器 | 移取液体 |
| 离心管 | 反应容器 |

## 4. 试剂配方

### 4.1 常用溶液配方

#### 4.1.1 磷酸盐缓冲液（PBS）
- **配方**：
  - 137 mM NaCl
  - 2.7 mM KCl
  - 10 mM Na₂HPO₄
  - 1.8 mM KH₂PO₄
  - pH 7.4
- **配制方法**：
  1. 称取 8 g NaCl，0.2 g KCl，1.44 g Na₂HPO₄，0.24 g KH₂PO₄
  2. 溶解于 800 mL 去离子水中
  3. 用 HCl 调节 pH 至 7.4
  4. 定容至 1 L
  5. 高压灭菌，4°C保存

#### 4.1.2 胰蛋白酶溶液（0.25%）
- **配方**：
  - 0.25% 胰蛋白酶
  - 0.02% EDTA
  - PBS 缓冲液
- **配制方法**：
  1. 称取 0.25 g 胰蛋白酶和 0.02 g EDTA·2H₂O
  2. 溶解于 100 mL PBS 缓冲液中
  3. 搅拌至完全溶解
  4. 过滤灭菌
  5. 分装后-20°C保存

#### 4.1.3 完全培养基（DMEM）
- **配方**：
  - 89% DMEM 基础培养基
  - 10% 胎牛血清 (FBS)
  - 1% 青霉素/链霉素 (100×)
- **配制方法**：
  1. 取 89 mL DMEM 基础培养基
  2. 加入 10 mL 胎牛血清
  3. 加入 1 mL 青霉素/链霉素 (100×)
  4. 混合均匀
  5. 4°C保存，使用前预热至37°C

#### 4.1.4 完全培养基（RPMI 1640）
- **配方**：
  - 89% RPMI 1640 基础培养基
  - 10% 胎牛血清 (FBS)
  - 1% 青霉素/链霉素 (100×)
- **配制方法**：
  1. 取 89 mL RPMI 1640 基础培养基
  2. 加入 10 mL 胎牛血清
  3. 加入 1 mL 青霉素/链霉素 (100×)
  4. 混合均匀
  5. 4°C保存，使用前预热至37°C

#### 4.1.5 细胞冻存液
- **配方**：
  - 70% 完全培养基
  - 20% 胎牛血清
  - 10% DMSO
- **配制方法**：
  1. 取 7 mL 完全培养基
  2. 加入 2 mL 胎牛血清
  3. 加入 1 mL DMSO
  4. 混合均匀
  5. 4°C预冷，现配现用

#### 4.1.6 台盼蓝溶液（0.4%）
- **配方**：
  - 0.4% 台盼蓝
  - PBS 缓冲液
- **配制方法**：
  1. 称取 0.4 g 台盼蓝
  2. 溶解于 100 mL PBS 缓冲液中
  3. 搅拌至完全溶解
  4. 过滤灭菌
  5. 4°C保存

## 5. 实验步骤

### 步骤1：细胞复苏

- **目的**：从液氮中取出冻存的细胞，恢复其生长能力
- **操作**：
  1. **准备工作**：
     - 预热培养基至37°C
     - 准备无菌的培养瓶和离心管
  2. **解冻细胞**：
     - 从液氮罐中取出细胞冻存管
     - 迅速放入37°C水浴中，轻轻摇晃使其快速解冻
     - 当冻存液融化至约1mL时，取出冻存管
  3. **细胞接种**：
     - 用75%乙醇消毒冻存管外壁
     - 将细胞悬液转移到含有预热培养基的离心管中
     - 1000rpm离心5分钟，弃上清
     - 用新鲜培养基重悬细胞
     - 将细胞接种到培养瓶中
     - 放入CO2培养箱中培养
- **注意事项**：
  - 解冻过程应迅速，避免细胞损伤
  - 应使用预热的培养基
  - 应确保无菌操作

### 步骤2：细胞传代

- **目的**：当细胞生长到一定密度时，将其分成多个培养瓶，继续培养
- **操作**：
  1. **准备工作**：
     - 预热培养基和胰蛋白酶至37°C
     - 准备无菌的培养瓶
  2. **消化细胞**：
     - 用PBS洗涤细胞2次
     - 加入适量胰蛋白酶，覆盖细胞表面
     - 37°C孵育2-5分钟，直到细胞变圆
     - 加入含有血清的培养基终止消化
  3. **细胞计数**：
     - 轻轻吹打细胞，使其分散成单细胞悬液
     - 取少量细胞悬液与台盼蓝混合
     - 使用血细胞计数板计数细胞
  4. **接种细胞**：
     - 根据计数结果，将细胞接种到新的培养瓶中
     - 加入适量新鲜培养基
     - 放入CO2培养箱中培养
- **注意事项**：
  - 胰蛋白酶消化时间应适当，避免过度消化
  - 吹打细胞时应轻柔，避免损伤细胞
  - 应根据细胞生长速度调整传代比例

### 步骤3：细胞换液

- **目的**：更换培养基，提供新鲜的营养物质
- **操作**：
  1. **准备工作**：
     - 预热培养基至37°C
  2. **换液操作**：
     - 从培养箱中取出培养瓶
     - 吸去旧培养基
     - 用PBS洗涤细胞1次
     - 加入新鲜培养基
     - 放入CO2培养箱中继续培养
- **注意事项**：
  - 换液时应避免损伤细胞
  - 应根据细胞生长状态和培养基颜色变化决定换液频率

### 步骤4：细胞冻存

- **目的**：保存细胞株，延长细胞寿命
- **操作**：
  1. **准备工作**：
     - 准备冻存液（通常为培养基+10% DMSO+20%血清）
     - 准备无菌的冻存管
  2. **消化细胞**：
     - 按照传代的方法消化细胞
     - 离心收集细胞
  3. **冻存操作**：
     - 用冻存液重悬细胞，调整细胞浓度至1-5×10^6 cells/mL
     - 将细胞悬液分装到冻存管中
     - 放入程序降温盒中，-80°C过夜
     - 转移到液氮中长期保存
- **注意事项**：
  - DMSO对细胞有毒性，应在4°C预冷
  - 应使用程序降温，避免细胞损伤
  - 冻存管应做好标记，包括细胞名称、冻存日期等

### 步骤5：细胞计数与活力检测

- **目的**：计数细胞数量并检测细胞活力
- **操作**：
  1. **细胞悬液制备**：
     - 消化细胞，制备单细胞悬液
  2. **台盼蓝染色**：
     - 取10μL细胞悬液与10μL台盼蓝溶液混合
     - 室温孵育1-2分钟
  3. **计数**：
     - 将混合液加入血细胞计数板
     - 在倒置显微镜下计数活细胞（未染色）和死细胞（蓝色）
  4. **计算**：
     - 细胞浓度 = (四个大格细胞数/4) × 10^4 × 稀释倍数
     - 细胞活力 = (活细胞数/总细胞数) × 100%
- **注意事项**：
  - 台盼蓝染色时间不宜过长
  - 应计数足够数量的细胞以确保准确性
  - 应使用新鲜的台盼蓝溶液

## 5. 应用场景

- **基础研究**：研究细胞生物学、分子生物学等基本过程
- **药物筛选**：筛选潜在的药物分子
- **疫苗生产**：生产病毒疫苗
- **基因治疗**：研究和生产基因治疗载体
- **细胞治疗**：生产用于细胞治疗的细胞
- **毒性测试**：测试化合物的细胞毒性

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **贴壁细胞培养** | 细胞贴附在培养表面生长 | 大多数哺乳动物细胞 |
| **悬浮细胞培养** | 细胞在培养基中悬浮生长 | 某些细胞系，如淋巴细胞 |
| **三维细胞培养** | 细胞在三维环境中生长 | 模拟体内环境，研究细胞-细胞相互作用 |
| **无血清培养** | 使用无血清培养基 | 减少血清带来的变异性，用于蛋白生产 |
| **共培养** | 多种细胞共同培养 | 研究细胞间相互作用 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 细胞污染 | 操作不当或环境不清洁 | 严格无菌操作，定期检测污染 |
| 细胞生长缓慢 | 培养基营养不足或细胞老化 | 更换新鲜培养基，使用早期 passage 的细胞 |
| 细胞形态异常 | 培养条件不适或细胞分化 | 优化培养条件，检查细胞分化状态 |
| 细胞凋亡 | 培养条件不适或过度传代 | 优化培养条件，避免过度传代 |
| 支原体污染 | 血清或其他试剂污染 | 使用支原体检测试剂盒，定期处理 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可在体外研究细胞生物学过程 | 与体内环境存在差异 |
| 可控制实验条件，重复性好 | 技术要求高，需要专业设备 |
| 可用于大规模药物筛选 | 细胞可能发生遗传变异 |
| 可生产大量细胞用于研究 | 某些细胞类型难以培养 |
| 可用于基因编辑和细胞工程 | 成本较高，需要无菌环境 |
