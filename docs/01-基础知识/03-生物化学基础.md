# 03-生物化学基础

## 1. 定义与核心概念

### 1.1 生物化学 (Biochemistry)

**定义**：生物化学是研究生物体中化学物质和化学反应的学科，主要关注生物分子的结构、功能和代谢过程。

**核心概念**：
- **生物分子**：构成生物体的基本化学物质，如蛋白质、核酸、脂质和碳水化合物
- **代谢**：生物体中进行的化学反应的总和
- **酶**：生物体内的催化剂，加速化学反应
- **能量代谢**：能量的产生、储存和利用
- **信号传导**：细胞间和细胞内的信息传递

## 2. 生物分子

### 2.1 蛋白质 (Protein)

**结构层次**：
- **一级结构**：氨基酸序列
- **二级结构**：α-螺旋、β-折叠等
- **三级结构**：整条肽链的空间构象
- **四级结构**：多个亚基的组装

**功能**：
- 催化作用（酶）
- 结构支持（如胶原蛋白）
- 运输功能（如血红蛋白）
- 免疫保护（如抗体）
- 信号传导（如激素受体）

### 2.2 核酸 (Nucleic Acid)

**类型**：
- **DNA (Deoxyribonucleic Acid)**：遗传信息的主要载体
- **RNA (Ribonucleic Acid)**：参与基因表达和蛋白质合成

**结构**：
- 核苷酸组成：磷酸、五碳糖、含氮碱基
- DNA 为双链双螺旋结构
- RNA 多为单链结构

### 2.3 脂质 (Lipid)

**类型**：
- 脂肪（甘油三酯）
- 磷脂
- 固醇（如胆固醇）
- 蜡

**功能**：
- 能量储存
- 细胞膜组成
- 信号分子
- 保护和绝缘

### 2.4 碳水化合物 (Carbohydrate)

**类型**：
- 单糖（如葡萄糖、果糖）
- 双糖（如蔗糖、乳糖）
- 多糖（如淀粉、糖原、纤维素）

**功能**：
- 能量来源
- 结构支持（如纤维素）
- 细胞识别（如糖蛋白）

## 3. 酶学

### 3.1 酶的性质

- **催化作用**：降低反应活化能
- **特异性**：对底物有高度选择性
- **调节性**：受多种因素调控
- **高效性**：催化效率远高于化学催化剂

### 3.2 酶的分类

| 分类 | 反应类型 | 示例 |
|------|----------|------|
| 氧化还原酶 | 氧化还原反应 | 脱氢酶、氧化酶 |
| 转移酶 | 基团转移 | 激酶、转氨酶 |
| 水解酶 | 水解反应 | 蛋白酶、淀粉酶 |
| 裂合酶 | 化学键断裂 | 醛缩酶、脱羧酶 |
| 异构酶 | 异构化反应 | 变位酶、异构酶 |
| 连接酶 | 化学键形成（需ATP） | DNA连接酶、合成酶 |

### 3.3 酶的调控

- **别构调节**：效应物与酶的别构位点结合
- **共价修饰**：如磷酸化/去磷酸化
- **酶原激活**：无活性酶原转化为活性酶
- **反馈抑制**：代谢产物抑制酶活性

## 4. 代谢途径

### 4.1 糖代谢

- **糖酵解**：葡萄糖 → 丙酮酸，产生 ATP
- **柠檬酸循环**：丙酮酸 → CO₂ + H₂O，产生 NADH 和 FADH₂
- **氧化磷酸化**：NADH 和 FADH₂ → ATP，在线粒体中进行
- **糖异生**：非糖物质 → 葡萄糖

### 4.2 脂质代谢

- **脂肪分解**：脂肪 → 甘油 + 脂肪酸
- **β-氧化**：脂肪酸 → 乙酰 CoA，产生 ATP
- **脂肪酸合成**：乙酰 CoA → 脂肪酸
- **胆固醇合成**：乙酰 CoA → 胆固醇

### 4.3 氨基酸代谢

- **蛋白质降解**：蛋白质 → 氨基酸
- **氨基酸脱氨**：氨基酸 → α-酮酸 + 氨
- **尿素循环**：氨 → 尿素
- **氨基酸合成**：α-酮酸 → 氨基酸

### 4.4 核苷酸代谢

- **嘌呤合成**：从头合成和补救合成
- **嘧啶合成**：从头合成和补救合成
- **核苷酸降解**：嘌呤 → 尿酸，嘧啶 → β-丙氨酸等

## 5. 能量代谢

### 5.1 ATP (Adenosine Triphosphate)

- **结构**：腺苷 + 三个磷酸基团
- **功能**：细胞的能量货币
- **生成方式**：底物水平磷酸化和氧化磷酸化

### 5.2 电子传递链

- **位置**：线粒体内膜
- **组成**：复合物 I-IV 和辅酶 Q、细胞色素 c
- **功能**：将电子从 NADH 和 FADH₂ 传递给 O₂，产生质子梯度

### 5.3 氧化磷酸化

- **机制**：质子梯度驱动 ATP 合成酶合成 ATP
- **效率**：每分子葡萄糖约产生 30-32 分子 ATP

## 6. 信号传导

### 6.1 细胞外信号分子

- **激素**：如胰岛素、肾上腺素
- **神经递质**：如乙酰胆碱、多巴胺
- **细胞因子**：如白细胞介素、干扰素

### 6.2 信号传导途径

- **G 蛋白偶联受体 (GPCR)** 途径
- **受体酪氨酸激酶 (RTK)** 途径
- **离子通道受体** 途径
- **核受体** 途径

### 6.3 第二信使

- **cAMP (Cyclic Adenosine Monophosphate)**
- **cGMP (Cyclic Guanosine Monophosphate)**
- **IP₃ (Inositol Trisphosphate)** 和 DAG (Diacylglycerol)
- **Ca²⁺ (Calcium Ion)**

## 7. 生物膜与物质运输

### 7.1 生物膜结构

- **流动镶嵌模型**：磷脂双分子层 + 蛋白质 + 糖类
- **膜蛋白**：整合蛋白、外周蛋白、脂锚定蛋白
- **膜流动性**：影响细胞功能和信号传导

### 7.2 物质运输

- **被动运输**：简单扩散、易化扩散
- **主动运输**：原发性主动运输（如 Na⁺-K⁺ ATP 酶）、继发性主动运输
- **胞吞作用**：吞噬、胞饮、受体介导的内吞
- **胞吐作用**：分泌蛋白的释放

## 8. 生物化学技术

### 8.1 分离技术

- **电泳**：分离蛋白质和核酸
- **层析**：离子交换、凝胶过滤、亲和层析
- **离心**：分离细胞器和生物大分子

### 8.2 分析技术

- **分光光度法**：定量分析
- **质谱**：蛋白质和核酸分析
- **X 射线晶体学**：蛋白质结构测定
- **核磁共振 (NMR)**：分子结构分析

## 9. 生物化学与分子生物学的关系

### 9.1 相互联系

- 分子生物学研究基因的结构和功能，依赖生物化学技术
- 生物化学研究生物分子的代谢和功能，为分子生物学提供基础
- 两者共同构成现代生命科学的核心

### 9.2 研究热点

- **蛋白质组学**：研究细胞内所有蛋白质
- **代谢组学**：研究细胞内所有代谢物
- **系统生物学**：整合多组学数据，研究生物系统
