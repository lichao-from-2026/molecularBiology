# 一、分子生物学知识库

## 1. 项目简介

本项目是一个基于 RsPress 构建的分子生物学知识库，旨在为学习者提供从零开始的分子生物学学习路径和系统化的知识整理方案。

:::tip
本文档是知识库的**大纲与学习计划**，使用可交互的表格记录文档生成进度。点击状态即可切换：⬜ 待完成 → 🔄 进行中 → ✅ 已完成。
:::

## 3. 文档生成规范

根据 AGENTS.md 规定，每篇文档必须包含以下五个部分：

1. **定义与核心概念**
2. **分子机制/原理**
3. **实验方法与流程**
4. **应用场景**
5. **注意事项与常见误区**

:::tip
生成文档时，请先检查对应文件是否已存在，如已存在则进行更新，如不存在则创建新文件。
:::

## 4. 组件使用说明

本知识库使用了自定义的交互式状态表格组件 `<DocStatusTable />`，提供以下功能：

| 功能 | 说明 |
|------|------|
| **点击切换状态** | 点击任意行的状态Badge即可切换状态 |
| **展开详情面板** | 点击 ▶ 按钮展开详细列表 |
| **进度条显示** | 自动计算并显示完成百分比 |
| **本地存储** | 状态会保存到浏览器 localStorage |
| **统计显示** | 显示待完成/进行中/已完成的数量 |

### 0.1 组件参数

```typescript
interface DocStatusItem {
  id: string;
  title: string;
  description?: string;       // 简洁描述（折叠时显示）
  details?: string | string[]; // 详细内容（展开时显示）
  filePath?: string;
  status: 'pending' | 'in-progress' | 'completed';
}

interface DocStatusTableProps {
  items: DocStatusItem[];    // 文档列表
  title?: string;            // 表格标题
  storageKey?: string;       // localStorage 键名
}
```

### 0.2 状态说明

| 状态 | 符号 | 颜色 | 含义 |
|------|------|------|------|
| 待完成 | ○ | 灰色 | 尚未开始生成文档 |
| 进行中 | ◐ | 橙色 | 正在生成或修改文档 |
| 已完成 | ● | 绿色 | 文档已生成并定稿 |
