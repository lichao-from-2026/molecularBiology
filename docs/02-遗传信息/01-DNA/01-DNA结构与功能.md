# 01-DNA结构与功能

## 1. 定义与核心概念

### 1.1 DNA (Deoxyribonucleic Acid)

**定义**：DNA 是一种双链核酸分子，是遗传信息的主要载体，包含了生物体发育和功能所需的全部遗传指令。

**核心概念**：
- **核苷酸**：DNA 的基本组成单位，由磷酸、脱氧核糖和碱基组成
- **双螺旋结构**：DNA 的二级结构，由两条反向平行的多核苷酸链组成
- **碱基配对**：A-T、G-C 互补配对
- **半保留复制**：DNA 复制的方式，每条新链都包含一条母链

## 2. 分子结构

### 2.1 一级结构
- **组成**：由脱氧核苷酸通过磷酸二酯键连接而成的线性多聚体
- **碱基**：腺嘌呤 (A)、胸腺嘧啶 (T)、鸟嘌呤 (G)、胞嘧啶 (C)
- **方向**：5' → 3' 方向

### 2.2 二级结构
- **双螺旋**：两条反向平行的 DNA 链围绕共同的轴盘旋
- **氢键**：A-T 之间形成 2 个氢键，G-C 之间形成 3 个氢键
- **大沟和小沟**：双螺旋表面的沟槽，是蛋白质与 DNA 相互作用的位点

### 2.3 三级结构
- **超螺旋**：DNA 双螺旋进一步扭曲形成的结构
- **核小体**：DNA 缠绕组蛋白形成的基本结构单位
- **染色质**：核小体进一步折叠形成的结构
- **染色体**：细胞分裂时染色质高度凝聚形成的结构

## 3. DNA 复制

### 3.1 复制机制
1. **起始**：识别复制起点，解旋酶解开双螺旋
2. **延伸**：DNA 聚合酶合成新链
3. **终止**：复制叉相遇，复制结束

### 3.2 关键酶和蛋白

| 酶/蛋白 | 功能 |
|---------|------|
| 解旋酶 | 解开 DNA 双螺旋 |
| 单链结合蛋白 | 稳定单链 DNA |
| 引物酶 | 合成 RNA 引物 |
| DNA 聚合酶 III | 主要的 DNA 合成酶 |
| DNA 聚合酶 I | 切除 RNA 引物，填补缺口 |
| DNA 连接酶 | 连接 Okazaki 片段 |
| 拓扑异构酶 | 缓解复制过程中的超螺旋张力 |

### 3.3 半保留复制
- **特点**：每条新合成的 DNA 分子都包含一条母链和一条新链
- **验证**：Meselson-Stahl 实验

## 4. DNA 修复

### 4.1 修复类型

1. **直接修复**：直接修复损伤的碱基
2. **切除修复**：
   - 碱基切除修复
   - 核苷酸切除修复
   - 错配修复
3. **重组修复**：通过同源重组修复双链断裂
4. **易错修复**：SOS 修复系统

### 4.2 修复机制

| 修复类型 | 损伤类型 | 修复机制 |
|----------|----------|----------|
| 碱基切除修复 | 单个碱基损伤 | 糖苷酶切除损伤碱基，DNA 聚合酶填补 |
| 核苷酸切除修复 | 嘧啶二聚体等 | 切除包含损伤的寡核苷酸片段 |
| 错配修复 | 复制错误 | 识别并修复错配碱基 |
| 双链断裂修复 | DNA 双链断裂 | 同源重组或非同源末端连接 |

## 5. DNA 的功能

### 5.1 遗传信息存储
- **基因**：编码蛋白质或 RNA 的 DNA 片段
- **基因组**：生物体的全部遗传信息

### 5.2 基因表达调控
- **启动子**：RNA 聚合酶结合位点
- **增强子**：增强基因转录的 DNA 序列
- **沉默子**：抑制基因转录的 DNA 序列

### 5.3 复制和遗传
- **细胞分裂**：DNA 复制后平均分配到子细胞
- **减数分裂**：产生配子，实现遗传重组

## 6. 应用

### 6.1 基因工程
- **重组 DNA 技术**：将目的基因插入载体
- **转基因技术**：将外源基因导入生物体

### 6.2 医学诊断
- **基因检测**：检测基因突变
- **病原体鉴定**：检测病原体 DNA

### 6.3 法医学
- **DNA 指纹**：个人身份识别
- **亲子鉴定**：确定亲子关系
